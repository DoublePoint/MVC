GridRow
说明
AjaxDataGrid的子标签。如果不与AjaxDataGrid配套使用会出现模板混乱的情况。
属性
参数 说明 类型 可选值 默认值
field 该列绑定的数据源的字段名 String - -
title 该列对应的表头名 String - 默认为空字符串
width 列宽 数值 0-∞ 80
sort 该列是否可以排序 boolean true,false false
fixed 是否固定该列，当为true时左右拖动表格，该列始终显示 boolean true,false false
事件
事件名 说明 参数
onclick 鼠标单击列事件 未完善，待续...
函数
方法名 方法说明 参数 参数说明
addRecord 向表格添加数据 HyRecord,index HyRecord,index参数可以同时有，也可以都没有，或只有其中的一个)
详细使用情况

<ll-gridrow
field="cdmc"
title="菜单名称"
width="120"
fix="false"
sort="false"
></ll-gridrow>
