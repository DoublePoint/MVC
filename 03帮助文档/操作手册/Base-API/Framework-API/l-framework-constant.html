framework-constant.js
说明
定义了框架所用到的一些常量，例如表格的列默认宽度常量、树结构是否显示连接线等
属性
常量名 说明 类型 可选值 默认值
ConstantAjaxDataGrid：关于AjaxDataGrid的一些常量
DEFAULT_COLUMN_WIDTH AjaxDataGrid默认列宽 int 180
DEFAULT_COLUMN_SORT AjaxDataGrid默认列是否排序 boolean true、false false
DEFAULT_COLUMN_FIXED AjaxDataGrid默认列是否固定 boolean true、false false
DEFAULT_MIN_HEIGHT AjaxDataGrid默认行的最小高度 int 100
ConstantLayoutArea：关于栅栏LayoutArea的一些常量
DEFAULT_MIN_HEIGHT LayoutArea的默认最小高度 int ConstantAjaxDataGrid.DEFAULT_MIN_HEIGHT,默认取值为表格的默认最小高度
ConstantComponentMap：关于自定义标签的一些常量
_AjaxDataGrid 自定义标签（AjaxDataGrid）的默认标签名称 string ll-ajaxdatagrid
_AjaxDataGridRow 自定义标签（AjaxDataGridRow）的默认标签名称 string ll-gridrow
_AjaxMenu 自定义标签（AjaxMenu）的默认标签名称 string ll-menu
_AjaxForm 自定义标签（AjaxForm）的默认标签名称 string ll-ajaxform
_Tree 自定义标签（Tree）的默认标签名称 string ll-tree
_LayOut 自定义标签（LayOut）的默认标签名称 string ll-filllayout
_LayOutFieldH 自定义标签（LayOutFieldH）的默认标签名称 string ll-fillarea
_AjaxMenu 自定义标签（AjaxMenu）的默认标签名称 string ll-menu
ConstantPageDefaultParam：关于页面请求参数的一些常量
_IS_DO_INIT_FUNTION 请求地址的参数名，是否调用页面中的init方法 int _is_do_init_function
ConstantState：关于常用状态的一些常量
_YES_NO_STATE_Y 请求地址的参数名，是否调用页面中的init方法 string Y
_YES_NO_STATE_N 请求地址的参数名，是否调用页面中的init方法 string N
